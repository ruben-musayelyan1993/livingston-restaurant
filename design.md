# Design Document — Livingston Restaurant Landing Page

## Общая концепция

Философия ресторана — свобода (отсылка к «Чайке Джонатану Ливингстону»). Дизайн должен передавать ощущение высоты, простора и утончённости. Визуальный язык: роскошь без кричащей помпезности, сдержанная элегантность с живым теплом джазового вечера.

**Настроение:** тёмный элегантный зал ресторана, тёплый свет свечей, золото приборов, панорама ночного Еревана за стеклом, звуки джаза.

## Типографика

### Шрифты

**Заголовки:** [Cormorant Garamond](https://fonts.google.com/specimen/Cormorant+Garamond)  
Классический высококачественный serif, передаёт ощущение haute cuisine и изысканности. Используется в вариантах Light (300), Regular (400), SemiBold (600), Italic.

**Тело и UI:** [Montserrat](https://fonts.google.com/specimen/Montserrat)  
Современный гуманистический sans-serif, контрастирует с serif-заголовками. ExtraLight (200), Light (300), Regular (400), Medium (500).

**Декоративные акценты:** [EB Garamond](https://fonts.google.com/specimen/EB+Garamond) Italic  
Для цитат, подписей, коротких поэтических вставок.

### Шкала размеров

| Элемент | Размер Desktop | Размер Mobile | Вес | Семейство |
|---|---|---|---|---|
| Hero заголовок (H1) | 80–100px | 48px | 300 (Light) | Cormorant Garamond |
| Заголовок секции (H2) | 52px | 36px | 400 | Cormorant Garamond |

| Кнопки / метки | 12px | 12px | 500, letter-spacing: 0.15em | Montserrat uppercase |

### Правила набора

- Letter-spacing для uppercase UI-элементов: `0.1em–0.2em`
- Line-height для заголовков: `1.05–1.15`
- Line-height для текста: `1.7–1.85`
- Максимальная ширина строки (measure): `65–70 символов`

## Секции — детальный дизайн

### Hero

**Высота:** 100vh (полный экран)

**Фон:**  
Полноэкранная фотография — вид на ночной Ереван с балкона 9-го этажа, или крупный план эталонного блюда с боке-эффектом. Поверх — тёмный градиентный overlay (`hero-overlay`). Лёгкий parallax при скролле (background-attachment: fixed на desktop).

**Контент (центрировано вертикально и горизонтально):**
```
[логотип — svg, белый, ~160px ширины]

[тонкая золотая линия-разделитель, 60px]

LIVINGSTON
[Cormorant Garamond Light, ~90px, letter-spacing: 0.08em, цвет: #F5F0E8]

European Haute Cuisine · Yerevan
[Montserrat Light, 13px, letter-spacing: 0.25em, uppercase, цвет: #C9A96E]

[gap 48px]

[CTA-кнопка: «ЗАБРОНИРОВАТЬ СТОЛ»]
```

**CTA-кнопка:**  
Стиль: outline (ghost button). Рамка `1px solid #C9A96E`, цвет текста `#C9A96E`, фон прозрачный. При hover: фон заполняется `#C9A96E`, текст меняется на `#0A0A0A`. Transition: `0.3s ease`. Padding: `16px 40px`. Letter-spacing: `0.15em`. Font-size: `11px`, uppercase.

**Scroll indicator:**  
Внизу — анимированная стрелка вниз или текст «Scroll» с тонкой линией.

## Анимации и переходы

### При загрузке страницы
- Логотип и заголовок Hero появляются с fade-in + небольшим slide-up (translateY: 20px → 0)
- Delay: логотип 0ms, заголовок 200ms, подзаголовок 400ms, CTA 600ms

### При скролле (Intersection Observer)
- Каждый контентный блок появляется с `opacity: 0; transform: translateY(24px)` →  
  `opacity: 1; transform: translateY(0)`, duration `0.7s ease-out`
- Карточки блюд: staggered появление с задержкой 100ms между каждой

### Hover-состояния
- Все интерактивные элементы: transition `0.25–0.35s ease`
- Навигационные ссылки: underline-анимация снизу (pseudo-element width: 0 → 100%)
- Карточки блюд: subtle scale + image zoom

### Параллакс
- Hero: background-attachment: fixed (CSS parallax) на desktop
- Секция Атмосфера: аналогично

### Переключение языка
- Смена текста: короткое fade (opacity 0.3s) без перезагрузки

## Текстуры и материалы

- **Film grain:** SVG-фильтр или CSS noise, opacity 3–5% поверх тёмных фонов — добавляет глубину, убирает «пластиковость» плоских тёмных блоков
- **Золотые разделители:** `<hr>` или `::before`/`::after` pseudo-elements, ширина 60–80px, высота 1px, цвет `#C9A96E`, opacity 0.5
- **Рамки-акценты фотографий:** сдвинутая золотая рамка (offset ~12px) создаёт эффект глубины без тяжёлых теней

## Мобильный дизайн (< 768px)

- Форма: однo поле в строку

## Примеры визуального настроения (reference)

Ориентиры по стилю (НЕ по содержанию):
- Сайты ресторанов: **Eleven Madison Park**, **Le Bernardin**, **Noma**
- Тёмная элегантность: **The Macallan Whisky**, **Moët & Chandon**
- Типографика: **Aesop** (баланс serif/sans, много воздуха)

**Ключевые принципы:**
- Больше воздуха, чем контента
- Фотографии говорят сами за себя — минимум текста
- Золото — акцент, а не заполнитель
- Каждая секция должна «дышать»
